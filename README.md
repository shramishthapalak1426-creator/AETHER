# AETHER

AETHER is an AI-powered sentiment analysis dashboard that analyzes YouTube comments to understand audience mood using Natural Language Processing.

## Features
- YouTube comment sentiment analysis
- Mood visualization with charts
- Word cloud generation
- Emotion score meter
- Conversational chatbot
- Feedback system
- Light/Dark theme UI

## Tech Stack
- Python
- Flask
- HTML / CSS / JavaScript
- SQLite
- NLP (NLTK)
